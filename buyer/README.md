# TrustLink Buyer Platform

**About**

This is the TrustLink Buyer Marketplace application, a premium e-commerce platform for Benin-Nigeria trade.

**Development**

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`

**Tech Stack**

- React
- Vite
- Tailwind CSS
- TanStack Query
- Framer Motion
