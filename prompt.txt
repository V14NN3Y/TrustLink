## CONTEXTE & OBJECTIF
Construis **TrustLink**, une marketplace e-commerce transfrontalière entre le **Bénin** et le **Nigeria**.
- Les **acheteurs** sont béninois, ils paient en **Franc CFA (XOF / FCFA)**
- Les **vendeurs** sont nigérians
- Le modèle de confiance repose sur un système **Escrow** :
  Acheteur paie → Plateforme sécurise les fonds → Vendeur livre au Hub TrustLink → Acheteur reçoit → Vendeur est payé
- Livraison express **2 à 7 jours**
- Aucun backend réel : toutes les données sont **mockées** (localStorage pour persistance)

### Règles techniques absolues :
- `BrowserRouter` avec `basename={__BASE_PATH__}` dans `App.tsx` (__BASE_PATH__ injecté par Vite define)
- `AppRoutes` importé depuis `src/router/index.ts` — JAMAIS redéfini dans App.tsx
- `element` dans RouteObject = JSX `<Component />`, jamais `lazy()` direct
- Imports cross-dossier : `@/` (alias src/) — imports même dossier : `./`
- Jamais de `window.location.href` pour naviguer — utiliser uniquement `useNavigate` / `<Link>`
- Jamais de `require()` — ES modules uniquement
- Jamais de `React.Fragment` — utiliser `<></>`
- Fichiers maximum : **500 lignes**
- Mock data dans `src/mocks/*.ts` : export nommé uniquement, pas de default export, pas de fonctions, pas de données dynamiques
- Icônes : classes CSS directement (`<i className="ri-home-line"></i>`) — jamais d'import npm
- Images : URL Stable Diffusion `https://readdy.ai/api/search-image?query={prompt 150+ chars}&width={w}&height={h}&seq={unique_id}&orientation={landscape|portrait|squarish}` — URL hardcodée, jamais de template literals

## DESIGN SYSTEM
### Palette de couleurs (STRICTE)

brand.primary = #125C8D → CTA secondaires, search button, borders focus, liens actifs
brand.dark = #0E3A4F → Footer background, hero overlay, texte logo
brand.light = #E1F0F9 → Fonds sections réassurance, bouton “Ajouter” par défaut
brand.orange = #FF6A00 → CTA principal, badges promo, accents, hover icons
neutral.dark = #111827 → Corps de texte principal
neutral.mid = #6B7280 → Sous-titres, métadonnées, prix barrés
white = #FFFFFF → Fond page, cards
section.bg = #F8FAFC → Fond image dans les cards produit

**INTERDIT** : Aucune couleur bleue autre que #125C8D. Violet totalement prohibé.
**Fond page** : blanc pur ou couleurs RGB toutes > 240 (très clair, très peu saturé). Jamais gris.
### Typographie

Font principale : Poppins → Titres, labels, boutons, navigation (weights: 400/500/600/700/800)
Font secondaire : Inter → Corps de texte, descriptions, métadonnées (weights: 300/400/500/600)


**Échelle typographique :**
- Hero H1       : text-3xl md:text-5xl, Poppins Bold 700, blanc
- H2 sections   : text-xl md:text-2xl, Poppins SemiBold 600
- Card title    : text-sm, Inter Medium 500
- Prix principal: text-base font-bold, Poppins, color #125C8D
- Prix barré    : text-xs line-through, Inter, color #6B7280
- Badge/label   : text-xs font-bold uppercase, Poppins
- Corps courant : text-sm, Inter Regular 400
### Arrondis

Cards produit → rounded-xl (12px)
Boutons principaux → rounded-full (pill)
Boutons secondaires → rounded-lg (8px)
Badges promo → rounded-full (pill)
Search bar → rounded-full (pill)
Icônes header → rounded-full (cercle)
Icônes réassurance → rounded-xl (12px)


### Ombres
ZERO shadow (shadow-* prohibé). Relief obtenu par :
- `border border-gray-100` sur les cards
- `hover:-translate-y-1` pour l'effet de survol
- Exception : `shadow-sm` uniquement sur le header au scroll
### Animations
- Cards : `hover:-translate-y-1 transition-all duration-300`
- Images cards : `group-hover:scale-105 transition-transform duration-500`
- Hero slides : `opacity-0/100 transition-opacity duration-700`
- Bouton "Ajouter" : feedback `bg-green-100 text-green-700` pendant 1500ms puis reset
- Badge support flottant : `animate-bounce`
- Header : `transition-all duration-300` → `shadow-sm` quand `window.scrollY > 10`

VI. UTILITAIRES (src/utils/format.ts)

export const formatPrice = (price: number): string =>
  new Intl.NumberFormat('fr-FR').format(price) + ' FCFA';
// Résultat : "28 500 FCFA"
export const renderStars = (rating: number): string[] => {
  const full  = Math.floor(rating);
  const half  = rating % 1 >= 0.3 ? 1 : 0;
  const empty = 5 - full - half;
  return [
    ...Array(full).fill('ri-star-fill'),
    ...Array(half).fill('ri-star-half-line'),
    ...Array(empty).fill('ri-star-line'),
  ];
};
export const generateTracking = (): string =>
  'TL-' + Math.random().toString(36).substring(2, 10).toUpperCase();
